# V我50 文案机

语言：[English](README.md) | [简体中文](README.zh-CN.md)

给群聊生成一条新的 V50 小作文。

输入一个可选关键词，点击生成；不满意就点再来一条，直到出现一条顺眼的。觉得可以，就复制发出去。页面会保留最近 5 条结果，方便你挑一条最像样的。

这是一个同人梗工具，不是 KFC 官方产品，也不使用官方品牌素材。

## 你可以做什么

- 加一个关键词来控制铺垫方向。
- 一次生成一条短 V50 文案。
- 点再来一条，换一个角度。
- 把当前结果复制到剪贴板。
- 在页面里回看最近 5 条生成结果。

## 如果你想知道它怎么写的

它不会实时上网搜索。项目里内置了一批 V50 示例语料，这些内容会存进 SQL 数据库，并在向量数据库里建立索引。

当你输入关键词时，页面会用 BGE-M3 把关键词变成向量，再去向量索引里找语义接近的示例。挑参考时还会加一点多样性，避免拿到的全是同一个笑话的六种写法。所以它不是纯随机抽样。

最后真正写文案的是 Kimi K2.5。模型会看到你的关键词、选出来的参考，以及最近几条上一版结果，然后写出一条新的文案。

你点再来一条时，前几次会尽量贴着同一组参考，保持主题不跑偏。多点几次后，它会扩大搜索范围，让下一条更不容易被猜到。

当你点击复制时，这条结果会被记录成一条被接受的输出。页面上能看到的最近历史则只保存在你的浏览器本地。
